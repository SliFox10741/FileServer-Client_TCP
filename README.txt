# Клиент-Серверное Приложение на C#

## Описание
Это клиент-серверное приложение написано на языке C# и предназначено для обмена файлами между сервером и клиентом по TCP. Сервер может одновременно обрабатывать несколько клиентов. Клиент запрашивает файл у сервера, а сервер отправляет файл, если он найден в указанной директории.

## Структура
Проект состоит из следующих основных файлов:

- **Server.cs**: Определяет класс `Server`, который слушает входящие TCP соединения, принимает запросы на файлы и отправляет файлы клиентам.
- **Client.cs**: Определяет класс `Client`, который устанавливает соединение с сервером, запрашивает файлы и сохраняет их на локальной машине.
- **Program.cs**: Основной файл для запуска как сервера, так и клиента.

## Переменные и Константы

### Server.cs
- `Port`: Порт, на котором сервер слушает входящие соединения (5000).
- `filesPath`: Путь к директории, где сервер ищет запрашиваемые файлы.

### Client.cs
- `serverAddress`: Адрес сервера (по умолчанию 127.0.0.1, можно заменить на реальный IP или доменное имя).
- `Port`: Порт, на котором клиент подключается к серверу (5000).

## Как запускать программу
- Создать объект класса Server на сервере и объект класса Сlient у клиента при подключении
- Вызвать у объектов метод Start
